# 10 Songs to Get to Know Me

나를 알 수 있는 10곡의 노래를 소개하는 웹페이지입니다.

## 실행 방법

1. Python 내장 HTTP 서버를 실행합니다:
   ```bash
   python -m http.server 8000
   ```

   또는 Python 3를 사용하는 경우:
   ```bash
   python3 -m http.server 8000
   ```

2. 브라우저에서 다음 주소로 접속합니다:
   ```
   http://localhost:8000
   ```

3. 페이지에서 노래 목록을 확인하고 감상할 수 있습니다.

## 서버 종료

서버를 종료하려면 터미널에서 `Ctrl + C`를 누르세요.
